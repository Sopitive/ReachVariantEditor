player.get_button_time

a.k.a. player.get_button_press_duration

When called on a player, this function returns the amount of ticks that the player has been holding down a button up to a max of 255 ticks. Returns 0 if the player is not holding down the button.

This function returns number. Calling this function without storing its return value in a variable is an error.

Arguments

button

A button -- not a specific hardware button, like Left Bumper, but rather the button to which a given gameplay function is currently mapped.

jump
The button used to make the player jump
switch_grenade
The button used to switch grenades between frags/plasmas
switch_weapon
The button used to switch weapons between primary/secondary
context_primary
The button used as the "action" button to interact with objects
melee
The button used to make the player melee
equipment
The button the player uses to activate their armor ability
throw_grenade
The button used to throw a grenade
fire_primary
The button used to fire the primary weapon
crouch
The button used to make the player crouch
scope_zoom
The button used to toggle zooming in/out
night_vision
The button used to toggle night vision
fire_secondary
The button used for vehicle function 1
fire_tertiary
Appears to always trigger on the "B" button
vehicle_trick
The button used for vehicle function 2 and vehicle function 3

Notes

  • This function only appears to do anything when called on the host.

  • This function appears to only work when the player is using a controller.

  • When the player is holding down any button, this function will get the button time regardless of whether the button would normally be used in the current context. For example, pressing the jump button while in a vehicle will still return the amount of time the player has been holding down the jump button.
